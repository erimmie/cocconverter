html 코드 변환하기

1. 롤20 사이트 채팅 내역에서 ctrl + a 를 눌러 전체 셀렉 후 crtl + c 로 긁음!


2. 네이버 블로그로 가 '새 글쓰기'를 누르고 ctrl + v 로 붙여넣기 이후
우측 하단의 'html' 버튼 을 클릭해 html 코드로 변환.


3. 변환된 html 코드를 다시 ctrl + c 로 복사하여 본 사이트의 네모 박스에 붙여넣기 후 Convert 버튼 클릭!

p.s. ... 코드의 길이가 길면 (긴 세션일수록) 사이트가 터질 수 있습니다...
전체 코드를 몇 피스로 나누어 같은 과정을 여러 번 반복하셔야 할 수도...ㅜㅜ


=> 이 시점에는:
1. 파란색 지문(내가 작성한 지문)의 배경색이 다른 유저들의 지문과 같은 회색으로 변환됨
2. 화면을 뚫는 다이스 로그의 width 도 화면에 맞게(100%의 너비) 조정됨!

Convert 이후 Copy 버튼 을 클릭하면 변환된 코드가 클립보드에 복사됩니다.
이후 ctrl + v 를 통해 어디든 붙여넣기를 할 수 있습니다.

* 클릭 이후 이런 알림창이 뜨면 성공적으로 복사가 된 거예요. (사용하는 브라우저에 따라 다를 수 있음!!)

블로그에 백업하기


1. 티스토리 블로그로 이동해 우측의 드롭다운 메뉴에서 html 모드 선택

2. ctrl + v 로 코드 붙여넣기
(html모드에서 다시 기본 모드로 돌아오면 롤방처럼 보임! 띄엄띄엄 떨어져 보이지만 글을 발행하면 붙어서 보여요!)



3. 우측 하단의 완료 버튼을 눌러 포스트 발행으로 마무리.

이미지 url 변경하기

이렇게 백업된 포스팅들의 경우 roll20 에서 플레이어들이 라이브러리에 저장된 사진을 삭제할 경우 엑박으로 뜨게 됩니다,,
이런 현상을 방지하고자 이미지 url을 바꾸는 방법이 있습니다!

준비물:

roll20 에서 긁어온 원본 이미지 url

사전에 블로그 포스팅이나 imgur 등의 사이트를 이용해 generate한 새 url(이미지 주소)!
(저는 티스토리 블로그에 인장만 따로 올려놓고 url을 긁어오는 방식을 사용합니다!)


url 변환하기

1. 본 사이트로 돌아와 인풋 박스 위에 있는 url 칸들을 채워 줍니다.
좌측 인풋: roll20발 url | 우측 인풋: 새 url

2. 그리고 앞서 했듯이 convert 버튼을 누르는 것으로 마무리.
이후엔 html 코드 변환하기 섹션에서 했던 것과 똑같이 copy 버튼을 누르는 것으로 (이미지 url까지) 변환된 코드가 클립보드에 저장됩니다.
